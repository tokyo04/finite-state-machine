Finite State Machine (FSM) adalah metode untuk mengatur perilaku sistem dengan membaginya ke dalam beberapa keadaan (state),
di mana sistem hanya berada di satu state pada satu waktu dan berpindah ke state lain berdasarkan kondisi atau event tertentu.
